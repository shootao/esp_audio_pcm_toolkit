连接

PC 本机 IP（填到设备 menuconfig → Server IP）
★ 为系统推荐（默认路由）；多网卡时请选与 ESP 同网段的那个
TCP：「停止 Server」会断开连接；再次「启动 Server」后，设备会在下次发送 PCM 时自动重连（一般无需重启）。
UDP：「停止 Server」后再启动，设备无需重启，继续发包即可。
Web Serial mode: click Connect COM Port, pick USB Serial/JTAG in Chrome popup.

设备控制

连接后调整音量/增益，须点下方红色 Apply to device 才会下发（滑条不会自动发送）。

PCM 格式：多通道交错（Ch0, Ch1, … 按采样点排列），参数须与固件一致
网络模式：PC 作 Server，设备作 Client；menuconfig 填 PC IP + 端口（与上方监听端口一致）
增益命令：gch 0 30 = Ch0，与下方滑条 data-ch 一致（0 起）
波形操作：滚轮缩放 · Shift+滚轮平移 · 「跟随最新」回实时末尾
注意：USB 口应只发 PCM；Log/Console 请走其它串口，避免污染波形
